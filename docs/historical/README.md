# Historical / archival documents

This directory holds documents that were load-bearing earlier in the project and
have since been superseded. They are kept for audit, nostalgia, and context — not
as guides to current behavior.

**Treat these as museum pieces.** When they disagree with the current code or the
ratified decision records in `docs/decisions/`, the current state wins.

## Contents

- [`0000_original_plan.txt`](0000_original_plan.txt) — the day-one design plan
  (dated 2026-05-10) for AI Switchboard. The protocol, modes, and feature set
  described here predate the Conclave Charter, the decision-record process, and
  most of what actually shipped. Useful as a snapshot of the original intent;
  not useful as a guide to how the service behaves today.
